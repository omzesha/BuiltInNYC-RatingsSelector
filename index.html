How it works

Simple enough for today’s demo, structured for a real community/editor workflow.

1

Choose a rating

Set a ceiling such as G, PG, PG-13, R, or NC-17.

2

Match approved timestamps

The extension checks editor-approved ranges for the current video URL.

3

Mute or skip in the player

Only the selected categories are filtered, with a clear in-player indicator.

Demo-ready install

Load the repo as an unpacked Chrome extension, open the demo YouTube URL, and seek to the sample timestamps in the local database.

  1. Open chrome://extensions
  2. Turn on Developer mode
  3. Click Load unpacked
  4. Select this repository folder
